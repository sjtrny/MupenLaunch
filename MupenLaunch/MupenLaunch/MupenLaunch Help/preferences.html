Preferences

You can access the preferences by selecting Preferences from the MupenLaunch menu item: MupenLaunch > Preferences. Or by using the keyboard shortcut "⌘,".

Please note that changing the preferences will only take effect the next time you run the ROM.

Video

Here you can change the resolution that the emulation will run at and the video plugin to use.

The video plugin dictates how objects are rendered on the screen. You can choose the one that best suites your hardware and visual preferences.

Controllers

Here you can change key mapping settings, that means which keys, buttons or axis of a joystick map to Nintendo 64 controller buttons in the emulator.

Simply select the input option you would like to edit and in the table select a button or axis row. Then press the button or move the joystick on your controller to map them together.

Controllers

Here you can chose to use a custom Mupen64Plus executable.

Simply click on the drop down menu and select Custom. Click on the text field, this will bring up an open file dialog. Select your Mupen64Plus executable from the dialog and you are ready to go.

To change back use the drop down selector to Default.
