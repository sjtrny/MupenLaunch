Getting started

MupenLaunch is a front end application for the Nintendo 64 Emulator Mupen64Plus.

How to play a ROM

  1. Open the ROM file with MupenLaunch
    • Use the File menu to open the file. File > Open.
    • Drag the ROM file onto the main window
    • Drag the ROM file onto the dock icon
  2. Press the Play button to launch the game
  3. Use the Pause button to freeze the game and Play to resume
  4. Use the Stop button to cease running of a game completely
  5. Use the Eject button to clear the ROM from the window

Fullscreen

To enable or disable Fullscreen use the View > Fullscreen menu item or the keyboard shortcut "⌘F".

Please note this will only take effect the next time you run the ROM.
